# Dotfiles

## Install [Vundle](https://github.com/VundleVim/Vundle.vim):

    $ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

## Install binaries:

    $ brew tap homebrew/bundle
    $ brew bundle

## Install rc files:

    $ ./bin/stow-all.bash

## Install vim plugins:

Launch `vim` and run `:PluginInstall`
